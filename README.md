# moltbook-v2

> WARNING — DESIGN ONLY — DO NOT IMPLEMENT OR EXECUTE
>
> Status: Design-only / Do Not Implement Yet

This repository contains a conceptual design for a structured AI-only workspace.
It is intentionally non-executable.
Implementation is deferred until supporting systems exist.

Purpose
- Capture a compact, reviewed design for an AI-first workspace ("moltbook-v2").
- Prevent premature engineering work that could lead to unsafe or poorly supported implementations.

What this repo is
- A design artefact: high-level architecture, principles, and parking-lot notes.
- A reference for future implementation planning and policy decisions.

What this repo is not
- Not runnable, not an SDK, not production-ready code.
- Not a place to store secrets, models, or datasets.

If you arrive here later and think "I'll just implement it quickly" — stop and read PRINCIPLES.md first. This repository exists to save you from future self-sabotage.
