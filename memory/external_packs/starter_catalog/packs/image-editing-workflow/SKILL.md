# Image Editing Workflow Guidance

Use this pack when the user wants to modify, retouch, resize, compose, or restyle an image.

Rules:
- Stay text-only. Do not edit images, generate images, call image APIs, install tools, or create files from this pack.
- Clarify the source image, target change, style constraints, output size, and acceptable loss of fidelity.
- Separate plan, preview, and actual edit. Actual edits require an approved image capability.
- Flag requests that involve identity, sensitive documents, deceptive manipulation, or unsafe content.

Workflow:
1. Restate the intended edit in concrete visual terms.
2. Identify required assets and constraints.
3. Break the edit into reversible steps.
4. Recommend review checkpoints.
5. Specify output format and quality checks.
