# Personal Agent v0.2.5 Release Notes

v0.2.5 is a focused assistant UX patch on top of v0.2.4.

## Fixes

- Capability prompts now answer in normal user language.
- Internal architecture terms are hidden unless the user explicitly asks how
  the system works.
- Durable, temporary, sensitive, and uncertain memory requests are classified
  before storing.
- Useful durable memories can be recalled and forgotten.
- Temporary/low-value memories are not over-stored.
- Secret-like memory requests are refused.
- Vague requests get one concrete follow-up question.
- Mutating actions still explain the plan and require approval.

## Security

- Authorization, Plan Mode, Telegram token redaction, and mutation bypass
  protections are unchanged.
- Local inspection remains read-only.
- Secrets are not stored as assistant memory.

## Known limitations

- Memory classification is intentionally conservative and pattern-based for
  this patch.
- The assistant may ask for confirmation on ambiguous memory rather than
  guessing.
- Arbitrary malicious in-process Python is still not process-isolated.
